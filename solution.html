Our Solution:

Our team will use the current WearWare program to create a web API that can be accessed by authorized users. The API will return a JSON file that containing fields specified in the call ro our API. This will be able to include details such as calories, heart rate, sleep, METs, distance (steps), and physical activity level for specific participants participating in a study, or an entire study. Our team also plans to create internal logic to do analysis using OCTAVE. This analysis will reduce inconsistencies in the data and then send and SMS text message to study participants if certain criteria are met.
